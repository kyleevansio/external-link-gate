Warn visitors when opening third party links.

== Description ==

This plugin will show a confirmation modal when a visitor opens a third party link.

== Installation ==

1. Install the plugin through the Plugins section
1. Activate the plugin in the Plugins section
1. Configure settings through Settings > External Link Gate
